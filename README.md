## AirubbyUi

airubby-ui是自己工作中开发的一些组件库.

## 使用方法

1、Run`npm install airubby-ui --save`

2、在app.module.ts中引入airubby-ui
<pre>
import {AirubbyUIModule} from 'airubby-ui'
imports: [
    AirubbyUIModule
],
</pre>
